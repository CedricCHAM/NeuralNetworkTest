# Test reseaux de neurones

## Keras

*Utilisation de la bibliothèque* **Keras**

Le scripts necessite l'installation des bibliothèques **tensorflow** et **keras**

```shell
pip install keras
pip install tensorflow
```

**Lancement du script**

```bash
python test_keras.py
```
